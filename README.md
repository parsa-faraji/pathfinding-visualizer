# Pathfinding Visualizer

Interactive visualization of pathfinding algorithms. Draw walls, move start/end points, and watch algorithms find the shortest path.

## Algorithms

| Algorithm | Shortest Path? | Description |
|-----------|---------------|-------------|
| Dijkstra's | Yes | Explores by distance from start |
| A* Search | Yes | Uses heuristics for efficiency |
| BFS | Yes | Level-by-level exploration |
| DFS | No | Depth-first exploration |
| Greedy | No | Always moves toward goal |

## Features

- Drag start/end nodes
- Draw walls by clicking/dragging
- Generate maze patterns
- Adjustable animation speed
- Real-time statistics
